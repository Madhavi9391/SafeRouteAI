# SafeRoute AI

## Project Overview
AI-Based Accident Risk Prediction & Smart Route Safety System. 
Predicts accident probability for routes using weather, traffic, and time data. Suggests safest route instead of fastest.

## Application Flow
1. User registers/login (Email or Google OAuth)
2. Dashboard appears after login
3. User enters source & destination
4. Predict Risk calculates risk % and shows color-coded result

## Features Implemented
- Email registration + verification
- Google OAuth login
- Dashboard visible only after login
- Risk calculation based on weather, traffic, time
- Logout functionality

## Planned Features
- Real-time traffic integration
- Map visualization
- Mobile version

## Modules in Development (Beta)
- Smart route recommendation algorithm
- Accident pattern data integration

## Setup & Run Instructions
1. Clone the repository or download zip
2. Open `index.html` in a browser
3. Register/Login to access dashboard
4. Enter route details to see risk prediction
